# KoboDocs

Free invoice, receipt, quotation, and payslip generator for Nigerian businesses — plus budget planning, loan schedules, and an Ajo/Esusu contribution tracker. No login required.

## Status
Landing page preview — homepage only. Individual generator tools not yet built.

## Stack (planned)
- Cloudflare Pages
- Supabase (free tier, Pro accounts only)
- Paystack (payments)
